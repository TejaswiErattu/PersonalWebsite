# Tejaswi Erattu Taj Portfolio

This is the personal portfolio of Tejaswi Erattu Taj, showcasing skills, projects, and experience in software engineering and product development.

## Project Overview

The portfolio is built using Next.js and TypeScript, featuring a modern design with responsive components. It highlights key projects, work experience, and skills relevant to software engineering roles.

## Setup Instructions

1. Clone the repository:
   ```
   git clone https://github.com/TejaswiErattu/tejaswi-portfolio.git
   ```

2. Navigate to the project directory:
   ```
   cd tejaswi-portfolio
   ```

3. Install dependencies:
   ```
   npm install
   ```

4. Run the development server:
   ```
   npm run dev
   ```

5. Open your browser and visit:
   ```
   http://localhost:3000
   ```

## Project Structure

- **src/components**: Contains all React components for the portfolio.
- **src/data**: Holds structured data for projects, experience, and skills.
- **src/styles**: Includes global CSS styles.
- **src/pages**: Main entry points for the application.
- **src/types**: TypeScript types and interfaces for type safety.

## Features

- Responsive design
- Dynamic project and experience sections
- Contact form for inquiries
- Downloadable resume
